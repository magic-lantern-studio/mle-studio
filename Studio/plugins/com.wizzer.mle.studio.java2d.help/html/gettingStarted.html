Magic Lantern Quick Start Guide

This document discusses how to create a Magic Lantern title using the Magic Lantern Java2D Studio from Wizzer Works. These instructions assume that you are familiar with the Eclipse Integrated Development Environment or IDE upon which the Magic Lantern Studio is based. This version of the Magic Lantern Java2D Studio, or The Studio, is built upon version 4.16 of Eclipse.

The document first discusses how to launch The Studio using the Windows or Linux desktop shortcut. It then identifies how to create a project using a simple HelloWorld template. The guide will also address how to test and debug your application within the context of the IDE. Finally, this document presents a mechanism for bundling the Java2D application for distribution.
