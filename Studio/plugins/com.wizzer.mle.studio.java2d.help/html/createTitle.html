Creating a Magic Lantern Title

This document discusses how to create a Magic Lantern title using the Magic Lantern Studio SDK from Wizzer Works. The SDK is comprised of the Magic Lantern libraries and mastering tool chain.

Using the Magic Lantern Software Development Toolkit

Ideally, you should use the Magic Lantern Studio to create and modify a title. Refer to "Using the Magic Lantern Authoring Tools" for information on how to use the Magic Lantern authoring tools. However, you can create a title and add other actors to that title outside of the IDE.

There are some basic steps you take to create a title outside of The Studio. You have to

  1. create a Digital Workprint, DWP, file, this sets up the title workprint
  2. define actor definition files, .wpa and .wpd files
  3. set up a title build environment
  4. write the necessary Java or C++ code
